Get the game mode

Imagine your program is a game and it can accept a list of parameters, including game mode, game level, and so on.

If no parameters are passed, the default values are used.

Write a program that displays the value of the mode string parameter. If no parameter is provided, it should use the default string value default.

Note, that other parameters may be passed as well. You should only be concerned with the mode parameter.


Sample Input:
mode battle level one

Sample Output:
battle


Sample Input:
fullscreen true level one

Sample Output:
default



Memory limit: 256 MB
Time limit: 8 seconds
